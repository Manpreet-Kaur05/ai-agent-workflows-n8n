# AI Agent Workflows using n8n

This repository contains AI-powered automation workflows built using n8n, Gemini, APIs, and other AI tools.

## Projects

1. AI Weather Daily Planner
- Fetches weather data using API
- Generates personalized plans using Gemini
- Sends automated emails

2. AI Social Media Content Generator
- Summarizes articles
- Generates LinkedIn posts
- Creates AI-generated images
- Auto-posts to LinkedIn

3. AI Audio Summarization and Speech Generation
- Transcribes audio using Whisper
- Summarizes content using Gemini
- Converts text into speech

## Technologies Used
- n8n
- Google Gemini
- APIs
- Groq Whisper
- Murf AI
- Google Sheets
- LinkedIn API
